# 🗡️ Katana — Visual Experience

This is a small experimental project where I tried to create a cinematic web experience around the idea of a **katana** — not just as a weapon, but as a character.

🔗 Live: https://katana-char.vercel.app/

---

## ✨ Idea

The goal was to showcase the *presence* and *aura* of a katana through:

- Slow, cinematic reveals  
- Controlled lighting and shadows  
- Subtle particles, heat, and atmosphere  
- Scroll-based storytelling  

Instead of a typical website, this is more of a **visual journey**.

---

## 🎬 What it includes

- Scene-based transitions (scroll-driven)
- Blade reveal animation  
- Rotating and detailed product-style views  
- Atmospheric effects (particles, smoke, heat feel)  
- Final cinematic scene with character + dragon  

---

## 🤖 Built with AI-first approach

This project is special for me because:

- This is the **first time I tried building something using 100% AI-assisted workflow**
- Everything involved AI:
  - Image generation  
  - Video generation  
  - Scene ideation  
  - Code generation & iteration  

---

## 🧠 What I learned

- How to think in **scenes instead of pages**
- How to combine **visual storytelling + web development**
- Using **scroll as a narrative tool**
- Blending **AI + frontend engineering**

---

## 🚧 Status

This is an experimental build — not perfect, but a strong step toward building cinematic web experiences.
